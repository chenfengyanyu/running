如上所示，引用类型的数据，都返回了 object，我们无法做到精确判断。我们来总结一下：
1.对于基本类型，除 null 以外，均可以返回正确的结果。
2.对于引用类型，除 function 以外，一律返回 object 类型。
3.对于 null ，返回 object 类型。
4.对于 function 返回 function 类型。

这就需要用到 instanceof 来检测某个对象是不是另一个对象的实例。
