JS 数组操作汇总

2017/3月/28 19:19:51 发布在 技术博文 123 次阅读
   次阅读

JS 中对数组的操作并不是很多，但总是记不清楚。每每用到，需要翻查文档才能确定。索性整理整理，加强记忆，顺便巩固一下基础知识。

一、concat()

连接两个或更多的数组，并返回结果。

二、join()

把数组的所有元素放入一个字符串。元素通过指定的分隔符进行分隔。

三、pop()

删除并返回数组的最后一个元素

四、push()

向数组的末尾添加一个或更多元素，并返回新的长度。

五、reverse()

颠倒数组中元素的顺序。

六、shift()

删除并返回数组的第一个元素

七、slice()

从某个已有的数组返回选定的元素

八、sort()

对数组的元素进行排序

九、splice()

删除元素，并向数组添加新元素。

十、toSource()

返回该对象的源代码。

十一、toString()

把数组转换为字符串，并返回结果。

十二、toLocaleString()

把数组转换为本地数组，并返回结果。

十三、unshift()

向数组的开头添加一个或更多元素，并返回新的长度。

十四、valueOf()

返回数组对象的原始值
