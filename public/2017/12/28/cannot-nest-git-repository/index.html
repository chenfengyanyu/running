如何解决 Git 仓库嵌套问题

2017/12月/28 10:51:52 发布在 异常处理 123 次阅读
   次阅读

用了很久 Git ，今天却被 Git 嵌套问题卡住了。本文将围绕“如何解决 Git 仓库嵌套”问题展开讨论，并提供解决方案。

一、问题描述

Git 仓库嵌套使用后，被嵌套的 Git 仓库不能被外层 Git 仓库检测到。

这也就意味着你在被嵌套的 Git 仓库中的更改，外层仓库是无感知的。导致这个问题的原因可能如下：

  • 在代码中引入其他 Git 仓库；
  • Clone 别人的 Demo 并做修改；

不看场景的解决方案就是耍流氓。
