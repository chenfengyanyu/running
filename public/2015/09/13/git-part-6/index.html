git强大之处就是分支的快速切换，那么分支怎么理解呢？
git 中的分支，其实本质上仅仅是个指向 commit 对象的可变指针。

git 会使用 master 作为分支的默认名字。在若干次提交后，你其实已经有了一个指向最后一次提交对象的 master 分支，它在每次提交的时候都会自动向前移动。
